# **App Name**: DASH Decoded

## Core Features:

- Hero Section: Displays a welcome message with an animated icon and a call-to-action button to explore the interactive visualizer.
- Restaurant Analogy: Presents a comparison between traditional video streaming and DASH using a restaurant analogy to explain the adaptive streaming concept.
- Core Components Explanation: Details the three essential components of a DASH stream (MPD, Segments, Player) through descriptive info cards.
- Interactive DASH Visualizer: Integrates a live DASH player with real-time log outputs, buffer status, and network simulation controls for an interactive learning experience.
- Real-World Discovery Guide: Offers a step-by-step guide on how to find and examine DASH components on live streaming websites using browser developer tools.
- Key Benefits Summary: Summarizes the main advantages of DASH, focusing on reduced buffering, faster start-up, and optimal quality delivery.

## Style Guidelines:

- Primary color: Electric Purple (#BE72FF) to convey modernity and technological sophistication.
- Background color: Dark Gray (#212529) to provide a professional dark mode appearance.
- Accent color: Sky Blue (#72B6FF) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a clean and readable interface.
- Single-page layout with smooth scrolling between sections for an intuitive user experience.
- Use Font Awesome icons to visually represent key concepts and components of the DASH protocol.
- Subtle animations on the hero section icon and interactive visualizer to engage users and illustrate dynamic content.
